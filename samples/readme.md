This folder contains samples for DevOps security in Defender for Cloud.

## Contents

* [IaCMapping/](IaCMapping/) — Infrastructure as Code mapping with Terraform and an Azure DevOps pipeline.
* [trivypipeline/](trivypipeline/) — Azure DevOps pipeline that runs Trivy via `MicrosoftSecurityDevOps@1`.
* [copilotDemo/](copilotDemo/) — Pipeline and deployment sample used in Copilot demos.
* [checkovAzurePipelines/](checkovAzurePipelines/) — Verification fixture for the `CKV_AZUREPIPELINES_*` severity promotion (issue #164).
* [configs/](configs/) — Example `.gdnconfig` files for each supported tool.
